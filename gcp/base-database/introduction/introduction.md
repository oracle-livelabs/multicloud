# Introduction

Oracle AI Database@Google Cloud is a managed Oracle Database service that runs Oracle Cloud Infrastructure (OCI)–managed Oracle databases colocated inside Google Cloud data centers, giving Google cloud  applications low-latency access to Oracle Database services while billing and management are integrated through Google cloud.

Oracle Base Database Service on Oracle AI Database@Google Cloud delivers Oracle Database capabilities natively within Google Cloud, giving customers a simplified, cost-effective way to run Oracle workloads close to their Google Cloud applications and AI services. It combines the reliability and compatibility of Oracle Database with the operational simplicity of Google Cloud, enabling enterprises to modernize applications, reduce latency, and accelerate multicloud adoption without major refactoring.

The service supports Oracle Database 19c and Oracle AI Database 26ai, and integrates with Google Cloud services such as Vertex AI, BigQuery, and Gemini. Customers can provision and manage databases directly through Google Cloud tools and APIs while leveraging Oracle’s enterprise-grade automation, security, high availability, and AI-powered database features.

Oracle Base Database Service is designed for organizations that want the flexibility of Oracle Database in a multicloud architecture, offering an accessible entry point for mission-critical applications, transactional systems, and AI-driven workloads running across Oracle and Google Cloud environments.

## About this Workshop
In this workshop, you learn how to create A Base Database on Oracle AI Database@Google Cloud

Estimated Workshop Time: Two hours


### Objectives
In this workshop, you will create the following in **Google Cloud Console:**

* A VPC network
* An ODB Network
* An ssh key pair on a windows machine or a mac
* Base Database on Oracle AI Database@Google Cloud


### Prerequisites
- Access to Google Cloud Console, OCI tenancy, OCI console, and Oracle AI Database@Google Cloud
- An existing Google Cloud Account. Your Google Cloud account  must be onboarded and linked to an Oracle Cloud Infrastructure (OCI) tenancy.
- An Oracle Cloud Infrastructure (OCI) account with permissions to view and manage resources through the OCI console.


For detailed information about the prerequisites required to begin onboarding with `Oracle AI Database@Google Cloud` and to create Oracle Autonomous AI Database (Serverless) on `Oracle AI Database@Google Cloud`, see [Prerequisites](https://docs.oracle.com/en-us/iaas/Content/database-at-gcp/gcpcr-create-prerequisites.html).
