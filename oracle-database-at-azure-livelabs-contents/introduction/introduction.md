# Learn Oracle Database@Azure Architecture and Operations.

## Introduction

Oracle Database@Azure gives Azure and Oracle teams a shared path for Oracle database services. In this workshop, you review the Azure portal, the OCI control plane, Exadata infrastructure, Autonomous Database, and monitoring views.

Estimated Workshop Time: 75 minutes

### Prerequisites

- Access to an Azure subscription that contains an Oracle Database@Azure demo environment.
- Access to the paired OCI tenancy or a prepared screenshot-based classroom environment.
- Permission to view resource groups, virtual networks, Oracle Database@Azure resources, OCI subscription mappings, identity domains, databases, and metrics.
- Instructor guidance before any purchase, create, scale, or delete action.

### Objectives

- Explain how Azure subscription, resource group, and network views support Oracle Database@Azure.
- Locate Exadata infrastructure, VM clusters, DB servers, and databases from Azure and OCI.
- Describe where OCI manages subscription mapping, identity, scaling, backup, and recovery views.
- Review Autonomous Database capacity, backup, and disaster recovery settings.
- Use Azure and OCI metrics views to start an operational health discussion.

## Task 1: Understand the Workshop Flow

1. Review the workshop sequence.

    - Lab 1 starts in Azure with subscriptions, private offers, resource groups, and Oracle Database@Azure entry points.
    - Lab 2 inspects Exadata infrastructure views in Azure.
    - Lab 3 moves to OCI for subscription mapping, identity, VM cluster scaling, database backup, and monitoring views.
    - Lab 4 reviews Autonomous Database operations and the create workflow.
    - Lab 5 closes with monitoring and cost-management discussion points.

2. Treat each hands-on task as a guided review unless your instructor grants write access.

    - Do not purchase private offers during this workshop.
    - Do not create, scale, or delete resources in a shared demo tenancy.
    - Capture questions about naming, access, cost ownership, and recovery objectives for the final review.
