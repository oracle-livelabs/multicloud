# Workshop Details.

Estimated Time: 5 minutes

## Short Description

Learn how Oracle Database@Azure connects Azure portal workflows with OCI database views.

The workshop covers Exadata and Autonomous Database.

## Long Description

This workshop helps students understand Oracle workloads that run through Oracle Database@Azure.

Students review Azure subscription, private offer, resource group, and network views. They then inspect Exadata infrastructure and Autonomous Database resources.

The labs use a guided-review format. Students learn where Azure ends and where OCI control-plane actions begin.

They also review identity, scaling, backup, disaster recovery, monitoring, and cost ownership.

## Workshop Outline

- Introduction: workshop flow, prerequisites, and safety rules.
- Lab 1: Review Azure access, private offer, resource group, virtual network, and Oracle Database@Azure entry points.
- Lab 2: Review Exadata infrastructure, VM clusters, and DB servers in Azure.
- Lab 3: Explore OCI subscription mapping, administrator groups, VM cluster scaling, database backups, and monitoring.
- Lab 4: Review Autonomous Database operations and the create workflow.
- Lab 5: Review Azure and OCI metrics, then connect monitoring to cost-management follow-up.

## Prerequisites

- Azure portal access to a prepared Oracle Database@Azure demo subscription.
- OCI access to the paired tenancy or screenshot-based classroom materials.
- Read access to database resources, networks, identity views, and metrics.
- Instructor approval before any purchase, create, scale, or delete action.

## FreeSQL Summary

This workshop does not use FreeSQL. The source material teaches portal navigation, architecture review, and operational concepts rather than runnable SQL.
