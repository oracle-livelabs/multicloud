# Introduction

Oracle AI Database@AWS is a managed Oracle Database service that runs Oracle Cloud Infrastructure (OCI)–managed Oracle databases colocated inside AWS  data centers, giving AWS  applications low-latency access to Oracle Database services while billing and management are integrated through AWS.

Oracle Autonomous AI Database on Dedicated Exadata Infrastructure on Oracle AI Database@AWS brings Oracle’s fully managed, self-driving database to the AWS cloud—combining the power, security, and performance of Oracle Database with the operational simplicity and scalability of AWS-native deployments. Organizations can run mission-critical Oracle workloads on dedicated Exadata infrastructure hosted within AWS data centers, enabling low-latency integration with AWS services while benefiting from Autonomous AI Database capabilities such as automated patching, tuning, scaling, backup, and security. This solution helps enterprises accelerate cloud modernization, simplify database operations, reduce administrative overhead, and support high-performance transactional and analytical workloads with enterprise-grade availability and security—all while maintaining data residency and operational consistency across hybrid and multicloud environments.

**Dedicated Exadata performance and isolation** for enterprise-grade OLTP, analytics, AI, and mixed workloads.

**Full automation** of provisioning, tuning, scaling, patching, backups, and security operations.

**Native AWS integration** with services like EC2, IAM, CloudWatch, Redshift, Glue, Athena, Bedrock, and Amazon S3.

**Multicloud AI capabilities,** including AI Vector Search, Select AI, Retrieval-Augmented Generation (RAG), and in-database machine learning.

**Unified AWS procurement and operations**, allowing customers to consume Oracle database services through AWS Marketplace and existing AWS commitments.

## About this Workshop
In this workshop, you learn how to create an ODB Network, Exadata Infrastructure, Autonomous VM Cluster and finally an Autonomous AI Database.

Estimated Workshop Time: 6 to 7 hour


### Objectives
In this workshop, you will create the following in **AWS Console:**

* An ODB Network

* An Exadata Infrastructure

* An Autonomous VM Cluster

And you will create the following through **OCI  Console:**

* Autonomous Container Database 

* Autonomous AI Database 


### Prerequisites
- Access to AWS Console, OCI tenancy, OCI console, and Oracle AI Database@AWS.
- An existing AWS Account. Your AWS account  must be onboarded and linked to an Oracle Cloud Infrastructure (OCI) tenancy.
- An Oracle Cloud Infrastructure (OCI) account with permissions to view and manage resources through the OCI console.


For detailed information about the prerequisites required to begin onboarding with `Oracle AI Database@AWS` and to create Oracle Autonomous Database on Dedicated Exadata Infrastructure on `Oracle AI Database@AWS`, see [Prerequisites](https://docs.oracle.com/en-us/iaas/Content/database-at-aws/oaaws-prerequisites.htm).

You may now proceed to the next lab.
